### 2.5 进制转换

对一个**P进制的数**，如果要**转换为Q进制**，需要分两步：

1. 将 P进制数x **转换为 十进制数y**

   ```c++
   int y = 0, pro = 1;
   while(x != 0) {
       y += (x % 10) * pro;
       x /= 10;
       pro *= p;
   }
   ```

2. 将 十进制数y **转换为 Q进制数z**

   采用**除基取余法**

   ```c++
   int z[40], num = 0; //数组z 存放 Q进制数y 的每一位，num为位数
   do {
       z[num++] = y % Q; //除基取余
       y /= Q;
   } while(y != 0);
   ```

   使用`do···while`语句而不是`while`的原因是：**如果十进制数y = 0，使用`while`语句将使循环直接跳出，导致结果出错**。
